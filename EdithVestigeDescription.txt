Un clásico, refinado y mejorado, redescubre el personaje recortado de Antibirth: Edith

Edith: Vestige es un mod que toma la mecánica principal de Edith (Antibirth), hace unos cambios y la deja como una versión más presentable del personaje, siendo originalmente la idea de un desafío para el próximo mod Edith: Rebuilt, Vestige

- Control de salto: ¡Ten mayor control! Ahora el salto está asignado a un botón en vez de ser automático (tecla para SOLTAR)
- Interacción con stats: ¡Mejora tu pisotón con los stats! Ahora ciertos parámetros del pisotón dependerán de tus stats, aumenta tu área de efecto con el alcance o la fuerza de empuje con el shotspeed, entre otros
- Únicamente lágrimas: Edith será incapaz de usar otro tipo de ataques (Láseres, espadas, cuchillos, ¡Todo eso fuera!)
- Sinergias de pisotones: Aprovecha varios objetos para usar poderosas sinergias con los pisotones
- Potenciado por JumpLib y REPENTOGON: El mod hace uso de JumpLib para el manejo de los saltos y REPENTOGON para características que de otra forma serían imposibles de añadir

Detalles y cambios de Calidad de Vida misceláneos:
- Anulación de daño de pinchos 
- Menor empuje por parte de corrientes de agua
- Lágrimas de sal (efecto visual)
- Efectos visuales del pisotón mejorados respecto a Antibirth

Posibles preguntas:

¿Qué es Edith: Rebuilt?
Edith: Rebuilt es el proyecto original del que deriva Edith: Vestige, un mod cargado de más contenido, una contraparte Tainted y mucho más trabajado

¿Por qué decidiste hacer este mod?
Mientras trabajaba en el desafío, noté que la forma de controlar al personaje se posicionaba en medio del rework hecho para Rebuilt y lo trabajado en Antibirth, así que decidí hacer esto como un experimento

¿Cuándo se lanzará Edith: Rebuilt?
No hay una fecha exacta, Edith: Rebuilt requiere de funciones específicas de REPENTOGON+, el cuál se encuentra en fase de desarrollo y no es recomendable usar si no se es modder, una vez que se publique, se dará aviso de ello

¿Este mod tendrá una Tainted Edith?
No, la idea de este mod es ser una simple demostración de algo que vendrá en Rebuilt, el cuál SÍ incluye una Tainted Edith

¿Removerás Edith: Vestige de la workshop cuando se estrene Edith: Rebuilt?
No me he decidido sobre eso, pero seguramente mantendré ambas versiones subidas

¿Vestige seguirá siendo un desafío para Edith: Rebuilt?
Sí, el desafío se mantendrá igual y tendrá la misma jugabilidad que este mod

Edith: Rebuilt puede ser testeado ahora mismo descargándolo desde su repositorio de GitHub, tengan en cuenta que requiere de Repentance+ y REPENTOGON para funcionar adecuadamente

A classic, refined and improved, rediscover the cut character from Antibirth: Edith

Edith: Vestige is a mod that takes the main mechanics of Edith (Antibirth), makes some changes, and leaves it as a more presentable version of the character, originally intended as a challenge for the upcoming mod Edith: Rebuilt, Vestige

- Jump control: Have more control! Jumping is now assigned to a button instead of being automatic (RELEASE key)
- Tears only: Edith will be unable to use other types of attacks (Lasers, swords, knives, all that stuff is gone!)
- Interaction with stats: Improve your stomp with stats! Now certain stomp parameters will depend on your stats, increase your area of effect with range or thrust force with shotspeed, among others
- Stomp synergies: Take advantage of various objects to use powerful synergies with stomps
- Powered by JumpLib and REPENTOGON: The mod uses JumpLib for jump handling and REPENTOGON for features that would otherwise be impossible to add

Possible questions:

What is Edith: Rebuilt?
Edith: Rebuilt is the original project from which Edith: Vestige derives, a mod loaded with more content, a Tainted counterpart, and much more work.

Why did you decide to make this mod?
While working on the challenge, I noticed that the way the character was controlled was somewhere between the rework done for Rebuilt and the work done in Antibirth, so I decided to do this as an experiment.

When will Edith: Rebuilt be released?
There is no exact date. Edith: Rebuilt requires specific features from REPENTOGON+, which is currently in development and is not recommended for use if you are not a modder. Once it is released, I will announce it.

Will this mod feature a Tainted Edith?
No, the idea behind this mod is to be a simple demonstration of something that will be included in Rebuilt, which DOES feature a Tainted Edith.

Will you remove Edith: Vestige from the workshop when Edith: Rebuilt is released?
I haven't decided on that yet, but I will probably keep both versions uploaded.

Will Vestige continue to be a challenge for Edith: Rebuilt?
Yes, the challenge will remain the same and will have the same gameplay as this mod.

Edith: Rebuilt can be tested right now by downloading it from its GitHub repository. Please note that it requires Repentance+ and REPENTOGON to function properly.

Thank you for playing the mod, I hope you like it!
